3. 분석을 바탕으로 현재 디자인 시스템 토큰 최적화 + 커스텀 컴퍼넌트 (기존 재활용) 
4. 1번 기획 내용을 3번 결과에 맞게 작업할 템플렛 > 섹션 > 페이지 구현 계획
5. 최종 계획 및 각 컴퍼넌트 스토리북에 등록
6. 최종 완성 페이지 product에 반영
